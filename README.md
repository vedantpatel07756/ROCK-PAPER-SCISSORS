## Introduction

Welcome to the Rock Paper Scissors game! This is a simple web-based implementation of the classic hand game that you can play with your friends or solo.

## Features

- Classic Rock Paper Scissors gameplay.
- User-friendly interface with engaging visuals.
- Fun and easy way to decide winners and losers.
- Responsive design for various screen sizes.

## Demo

You can check out a live demo of the game  https://vedantpatel07756.github.io/ROCK-PAPER-SCISSORS/

## Installation

To run the game locally on your machine, follow these steps:

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/vedantpatel07756/ROCK-PAPER-SCISSORS.git
   ```

2. Navigate to the project directory:

   ```bash
   cd ROCK-PAPER-SCISSORS
   ```

3. Open the `index.html` file in your web browser.

## Usage

1. Choose your move: Rock, Paper, or Scissors.
2. The computer will also make a random choice.
3. The winner will be determined based on the rules: Rock beats Scissors, Scissors beats Paper, and Paper beats Rock.
4. Enjoy the game and have fun!
